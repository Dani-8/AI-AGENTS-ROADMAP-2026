- ✍️ **Prompt engineering** — few-shot examples, chain-of-thought, structured output (JSON mode)
- 🔧 **Tool / function calling** — giving an LLM the ability to call external functions and APIs
- 📚 **RAG (Retrieval-Augmented Generation)** — embeddings, vector stores, chunking strategies, retrieval quality
- 🔌 **First agent loops** — simple observe → think → act loops before any framework was involved
